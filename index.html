Tentang NiiHen App

Aplikasi mobile NiiHen ini memakai sistem webview, sehingga berukuran kecil dan tidak memerlukan izin khusus seperti kontak, sms, telepon. Aplikasi NiiHen saat ini diprogram untuk pengguna sistem operasi android, untuk sistem operasi iOS masih belum ada. Aplikasi NiiHen diprogram bekerja lebih efisien dengan browser Google Chrome, jadi NiiHen App akan sangat kompatibel jika sebelumnya pengguna telah menginstall browser Google Chrome.

Cara Mengoptimalkan?

Google Chrome

Install Google Chrome sebagai peramban default dan setting DNS Publik. Ini berfungsi supaya bisa membuka web yang terblokir dari provider internet di indonesia.

DNS Publik

Berikut cara setting DNS Publik Chrome:

buka chrome -> cari menu "setting/setelan" -> di bagian "dasar" ada menu "privasi dan keamanan/privacy and security" -> atau menu "gunakan DNS aman/use secure dns" -> pilih "penyedia lain/choose another provider" -> pilih "google (Public DNS)".

NiiHen App

Jalankan Aplikasi NiiHen. Jika setelah tombol "Buka NiiHen" dipencet tidak pindah apk, maka sekarang Aplikasi NiiHen sudah kompatibel dengan hp anda.
